# Life Calendar

Life Calendar est une application SaaS permettant de visualiser sa vie sous forme de grille temporelle. Elle offre une perspective unique sur le temps qui passe, permettant de marquer des événements marquants, des périodes de vie et de prendre conscience de chaque instant.

## Fonctionnalités

- **Visualisation Temporelle :** Vue par semaines, mois, ou blocs de jours (10, 100, 1000 jours).
- **Événements & Périodes :** Ajoutez des événements ponctuels ou des périodes (voyages, carrière, projets).
- **Calendriers Multiples :** Organisez vos événements par catégories (Santé, Travail, Personnel).
- **Personnalisation :** J0 basé sur votre date de naissance.

## Stack Technique

- **Framework :** Next.js 15 (App Router)
- **Langage :** TypeScript
- **Styling :** Tailwind CSS v4
- **Authentification :** Clerk (à venir)
- **Base de données :** PostgreSQL / Prisma (à venir)

## Installation

1.  Cloner le dépôt :
    ```bash
    git clone https://github.com/choudat/life-calendar.git
    ```
2.  Installer les dépendances :
    ```bash
    npm install
    ```
3.  Lancer le serveur de développement :
    ```bash
    npm run dev
    ```
